CSS

Cascading Style Sheets ou Folhas de estilo

Define como os elementos HTML são exibidos

Sintaxe

seletor {propriedade ou atributo:valor;}

h1 {color: red;}

Seletor

Seletor = qual elemento que será modificado

propriedade = o que vai ser modificado


Tipos de seletores

  • Simples: elemento, classe, id
  • combinados: dois ou mais tipos
  • outros: pseudo classes, pseudo elementos, seletores de atributos

Tipos de CSS

Inline

< style= "color:red;"< Isso é um parágrafo. <>

Interno

<style>

p {color: red;}

<style >

Externo

Html: <link rel= "stylesheet" href="estilo2.css">

p {color: red;}

Prioridades

Do menor para o maior:

  • ordem de declaração
  • elemento
  • classe
  • id
  • Especificidade da regra
  • !important
